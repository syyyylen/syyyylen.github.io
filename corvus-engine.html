Corvus Engine

This custom engine is focused on graphics features and has no vocation to become a game engine for now. It's a sandbox where I experiment with stuff I want to learn and implement on my own.

This engine is using DirectX 12 graphics API. I already had an engine using D3D11 and I wanted to learn D3D12 and modern rendering techniques.
It uses ImGui for editor UI and Assimp to load gltf files.
You can find the source code here.

Current features :

  • Deferred shading
  • PBR/IBL
  • Instancing
  • Point Lights
  • Lights Volumes
  • Shadow Maps
  • Water Rendering
    • Tesselation & Sum Of Sines Waves
    • Screen Space Reflections
    • Refraction Distortion
