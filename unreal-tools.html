UE Tools

Unreal Engine Plugins I've made to practice tools programming.

Console Commands Helper is a small plugin which simplifies the use of recurrent console commands.
It's available on the marketplace here, and since it's free I also have the repo in public here.
It can be a good starting point to learn tooling inside Unreal and Slate fundamentals.

Easy Colliders was a custom editor mode which allowed for fast placement of invisible walls in the level viewport. It was really fun to make and I've learned a lot of useful stuff, but it was a kinda bad solution to the problem it's trying to solve so it's not available anymore.
